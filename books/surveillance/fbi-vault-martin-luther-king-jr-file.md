# FBI Vault - Martin Luther King Jr. File

**Source:** FBI Vault

Declassified FBI surveillance files on Dr. Martin Luther King Jr., including wiretap authorizations, bug transcripts from hotel rooms, and the infamous anonymous letter urging King to commit suicide. Documents the FBI's Racial Matters program and Director Hoover's personal campaign to destroy King's reputation.
